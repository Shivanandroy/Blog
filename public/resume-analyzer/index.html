# define the ranking criteria that suits your requirement

search_criteria = {
    
    "Deep Learning": ["neural networks", "cnn", "rnn", "ann", "lstm", "bert", "transformers"],

}

ResumeAnalyzer passes this information to Spacy‚Äôs PhraseMatcher to calculate if that term is present in resume. If present, +1 is assigned to the resume in that category.

For e.g. ‚Äî if ‚ÄúNeural Network‚Äù is present in the resume, the candidates is assigned a +1 in the Deep Learning category.
